# Hygienic Air Rebuild

Modern Astro site rebuild for Hygienic Air, Inc.

## Development

- `npm run dev` - Start local development server
- `npm run build` - Build static site
- `npm run preview` - Preview production build

## Notes

- Site URL is configured in `astro.config.mjs`.
- Global SEO/meta/favicons/schema are defined in `src/layouts/SiteLayout.astro`.
- Modern site pages live in `src/pages/` and use `src/components/Site/`.
